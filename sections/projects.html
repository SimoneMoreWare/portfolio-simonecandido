Developed a Beta-VAE trained exclusively on LoS Channel Impulse Response data from the GEO-5G testbed at EURECOM. The model detects NLoS signals as out-of-distribution anomalies using the TISSI metric, achieving 88.97% accuracy (balanced) and 97.62% (realistic indoor conditions) — surpassing the 85% UWB benchmark. Deployed as a C++ xApp on the FlexRIC Near-RT RIC via LibTorch, processing 4-antenna CIR measurements in real-time (ms latency) with 99.7% NLoS detection on emulator testing.

Sviluppato un Beta-VAE addestrato esclusivamente su dati LoS di Channel Impulse Response dal testbed GEO-5G di EURECOM. Il modello rileva segnali NLoS come anomalie out-of-distribution tramite la metrica TISSI, raggiungendo 88.97% di accuratezza (bilanciato) e 97.62% (condizioni indoor realistiche) — superando il benchmark UWB dell'85%. Deployato come xApp C++ sul Near-RT RIC FlexRIC via LibTorch, elaborando misurazioni CIR a 4 antenne in tempo reale (ms di latenza) con 99.7% di rilevamento NLoS nei test con emulatore.

Built a monitoring system to migrate and analyze telemetry data from Kubernetes clusters. Engineered a Go backend with GORM and GJSON to extract data from AWS DynamoDB, transform it into a normalized relational schema (9 correlated tables), and load it into PostgreSQL via CloudNativePG. Implemented hybrid sync strategies (daily full refresh + incremental sync) handling DynamoDB's 1MB scan limit. Infrastructure orchestrated with Kubernetes/Kind/Docker. Designed advanced Grafana dashboards for cluster performance, geographical distribution, and versioning trends.

Sviluppato un sistema di monitoraggio per migrare e analizzare dati telemetrici da cluster Kubernetes. Backend in Go con GORM e GJSON per estrarre dati da AWS DynamoDB, trasformarli in uno schema relazionale normalizzato (9 tabelle correlate) e caricarli in PostgreSQL via CloudNativePG. Implementate strategie di sincronizzazione ibride (refresh completo giornaliero + sync incrementale) gestendo il limite di 1MB delle scan DynamoDB. Infrastruttura orchestrata con Kubernetes/Kind/Docker. Progettate dashboard Grafana avanzate per performance cluster, distribuzione geografica e trend di versioning.
